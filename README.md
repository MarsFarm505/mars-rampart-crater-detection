# Machine Learning Detection of Martian Rampart Craters

Mask R-CNN pipeline for detection and classification of Martian rampart craters using CTX imagery  
(Williams et al., Icarus, 2026)

---

## Overview
This repository contains the machine learning workflow used to detect and classify rampart craters on Mars.

The pipeline includes:
- CTX image preprocessing and tiling
- Mask R-CNN model training and inference
- Post-processing to remove duplicate detections
- Quantitative evaluation (confusion matrix)

## Model Files
The trained model includes:
- `.dlpk` – ArcGIS Pro deep learning package  
- `.emd` – model definition file  
- `.pth` – PyTorch model weights
